#  Growth Lever Effectiveness Framework

## 1. Measurement Methodology

To track if a growth lever (initiative) is successful, we use the following:

- **Baseline Metric**: Data point before implementation 
- **Target Lift**: Expected improvement 
- **Attribution Window**: Tracking impact for 60 days post-launch.

## 2. Success Indicators per Lever

- **Retention Lever**: 20% increase in 30-day user return rate.
- **Revenue Lever**: 10% conversion from free to pro pet records.
